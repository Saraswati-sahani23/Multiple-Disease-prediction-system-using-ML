# 🩺 Multiple Disease Prediction System

## 📌 Overview

The **Multiple Disease Prediction System** is a machine learning-based web application that predicts the likelihood of various diseases such as **Diabetes, Heart Disease, and Parkinson’s Disease** using user-provided medical data.

It provides quick and preliminary health insights through an easy-to-use interface, helping users understand potential health risks.

---

## 🚀 Features

* 🔍 Predicts multiple diseases in one platform
* 💻 User-friendly interface built with Streamlit
* ⚡ Fast and real-time predictions
* 📊 Uses trained machine learning models
* 🧠 Supports:

  * Diabetes Prediction
  * Heart Disease Prediction
  * Parkinson’s Disease Prediction

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Machine Learning:** Scikit-learn
* **Libraries:** NumPy, Pandas, Pickle

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/multiple-disease-prediction.git
cd multiple-disease-prediction
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the application

```bash
streamlit run app.py
```
